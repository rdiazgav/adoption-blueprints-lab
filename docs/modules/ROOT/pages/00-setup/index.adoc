= Workshop 00 вҖ” Platform Setup
:navtitle: Overview

This workshop prepares your environment and deploys the RetailFlow base application.
It is a *prerequisite for all other workshops*.

Estimated time: *30вҖ“45 minutes*

== What you will learn

* How to configure and verify Red Hat OpenShift Local (CRC)
* How to connect to an OpenShift cluster using the `oc` CLI
* How to deploy a multi-service application with Kustomize
* How to verify that all services are healthy using pod status, service endpoints, and a smoke test

== Steps

[cols="1,4",options="header"]
|===
| Step | Description
| xref:01-prerequisites.adoc[1 вҖ” Prerequisites] | Configure CRC or verify access to an external cluster
| xref:02-deploy.adoc[2 вҖ” Deploy RetailFlow] | Create the `retailflow` namespace and apply Kustomize overlays
| xref:03-verify.adoc[3 вҖ” Verify] | Confirm all pods are running, services are reachable, and the smoke test passes
|===
